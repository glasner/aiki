Tetris → Aiki mapping for ratatui:

Core metaphor:
• The well = the task workspace
• Pieces falling = subtasks being worked on
• Line clear = subtask completed
• Garbage lines from opponent = issues found during review (the killer feature of this metaphor)
• Clearing garbage = fixing issues
• Perfect clear (empty board) = clean commit
• Topped out = agent can't fix, escalates to human
• HOLD piece = next queued subtask
• NEXT piece = upcoming work
• Level = iteration number
• Lines = total subtasks completed

Why Tetris > other metaphors:
• Garbage lines — competitive Tetris sends garbage to your opponent. Review failures sending garbage to the board is exactly how it feels when issues get injected into your clean work. No other game metaphor has this.
• Clearing lines is satisfying — watching a full row flash and disappear maps perfectly to the dopamine of resolving an issue
• Board state = project health — a clean board means clean code. A messy board with gaps means tech debt. A topped-out board means escalation. The visual is immediately intuitive
• Tetris 99 for multi-task — multiple mini-boards side by side is a well-known visual pattern from Tetris 99/Tetris Battle
• Topped out = escalation — the game-over state naturally maps to "agent gives up, human takes over"

Tetromino colors (NES palette):
I=cyan, O=yellow, T=purple, S=green, Z=red, J=blue, L=orange

Garbage line colors by issue type:
• garb-red (#833) = errors (type, syntax)
• garb-mag (#846) = security issues
• garb-org (#863) = warnings (complexity, style)
Garbage lines have holes (the ·· gaps) — just like real Tetris garbage. The hole position is random, making each line clearable but requiring the right piece.

Characters: ██ (filled cell), ·· (empty cell), ▲ (garbage indicator on mini-board)

ratatui implementation: Each cell is 2 chars wide for square-ish proportions. The well is a 10-wide × 16-tall grid (20 chars × 16 rows). Mini-boards for Tetris 99 view are 5-wide × 6-tall. Use ratatui::widgets::canvas or raw Buffer writes for the block grid.
